## 🚀 Installation

1. Open **Extensions** sidebar in VSCode: `View → Extensions`
2. Search for `Void Iris`
3. Click **Install**
4. Reload the editor if prompted
5. Go to `File → Preferences → Color Theme → Void Iris`
6. Enjoy your journey into the void 🌌
